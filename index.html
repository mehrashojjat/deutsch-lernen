About

I built this free mini-game for myself — a simple way to practice German without any paywalls. I wanted it available as a free resource for anyone learning German.

Made by Mehras Hojjat. Feel free to reach out on LinkedIn or Instagram for feature requests or feedback.

Word lists from Goethe-Institut · Translations via MyMemory API · Open source on GitHub · Built with Claude AI.

Originally named DeutschLernen during early development.
